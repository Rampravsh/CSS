CSS Position Property Examples

position: static;

Static

This is the default position. The element is not positioned in any special way; it is always positioned according to the normal flow of the page.

position: relative;

Relative

This element is positioned relative to its normal position. With the properties top, right, bottom, and left, the element is moved from its normal position. Other content will not be adjusted to fit into any gap left by the element.

position: absolute;

This element is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).

Absolute

position: fixed;

Fixed

This element is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used.

position: sticky;

A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).

Sticky
